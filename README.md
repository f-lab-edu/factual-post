# factual-post
팩트인 정보 전달을 위한 프로젝트입니다. 

- 게시물 작성자는 올바른 정보 전달을 위해 질문이 달린 모든 글에 답변할 수 있어야 납득할 수 있는 글이 되고, 글 내용은 올바른 정보가 됩니다.
